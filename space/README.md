# Gerador de Poesias — Álvaro de Campos

Playground do modelo fine-tuned [`Rabramo/gerador-poesias-alvaro-campos`](https://huggingface.co/Rabramo/gerador-poesias-alvaro-campos).

Digite um verso inicial e o modelo gera um poema completo no estilo de **Álvaro de Campos**, heterônimo futurista de Fernando Pessoa.

## Configuração do token

Adicione `HF_TOKEN` nas variáveis secretas do Space:
**Settings → Variables and secrets → New secret → Name: `HF_TOKEN`**
